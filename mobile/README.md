# تطبيق الموبايل (Expo / React Native)

نسخة موبايل **أصلية** من الغرفة الصوتية مبنية بـ [Expo](https://expo.dev)
(React Native) تعمل على **Expo Go**. تتصل بنفس سيرفر الويب (Socket.IO + REST)
فتظهر نفس الغرف والمحفظة والهدايا والشات لحظياً.

> هذا مشروع منفصل تماماً عن `client/` (الويب) و`client/android/` (نسخة
> Capacitor). كلها تشترك في نفس السيرفر فقط.

## التشغيل على تلفونك (Expo Go)

1. ثبّت تطبيق **Expo Go** من App Store / Google Play.
2. على الكمبيوتر:
   ```bash
   cd mobile
   npm install      # أول مرة فقط
   npm start        # يشغّل Metro (خادم التطوير)
   ```
3. تأكّد أن **الهاتف والكمبيوتر على نفس شبكة الـWiFi**.
4. من Expo Go: امسح رمز QR الظاهر في الطرفية، أو اختر «Enter URL manually»
   واكتب: `exp://<IP-الكمبيوتر>:8081` (مثال: `exp://172.20.10.2:8081`).

يفتح التطبيق فوراً، وأي تعديل في الكود يتحدّث مباشرة (Fast Refresh).

## البنية

```
mobile/
├── App.js                    التنقّل: identity → lobby → room
├── app.json                  اسم التطبيق «الغرفة الصوتية» + ثيم داكن
└── src/
    ├── config.js             SERVER_URL (سيرفر Render الحقيقي افتراضياً)
    ├── theme.js              ألوان مطابقة للويب (بنفسجي داكن)
    ├── identity.js           uid ثابت + الاسم (AsyncStorage، نفس مفتاح الويب)
    ├── socket.js             اتصال Socket.IO
    ├── api.js                نداءات REST (الغرف/المحفظة/الهدايا)
    ├── screens/
    │   ├── IdentityScreen.js  إدخال الاسم
    │   ├── LobbyScreen.js     قائمة الغرف + المحفظة + الدخول بالرقم
    │   └── RoomScreen.js      المقاعد + المايك/الكتم + الشات + الهدايا
    └── components/
        └── GiftSheet.js       اختيار وإرسال هدية
```

## ما هو مُنفّذ

- ✅ هوية ثابتة مربوطة بنفس المحفظة على السيرفر (عبر `uid`).
- ✅ لوبي: قائمة الغرف الحيّة من `/api/rooms` + الدخول برقم غرفة.
- ✅ غرفة: 12 مقعداً، صعود/نزول عن المايك، كتم/فتح، مؤشر تحدّث.
- ✅ شات لحظي (Socket.IO) + رسائل النظام/الدخول.
- ✅ إرسال هدايا من الكتالوج الحقيقي مع كومبو، وخصم من المحفظة.
- ✅ عرض المحفظة (كوينز/ألماس) لحظياً.

## غير مُنفّذ بعد (قابل للإضافة)

- الصوت الحقيقي (WebRTC): Expo Go لا يدعم الوحدات الأصلية للصوت. يتطلب
  **Development Build** مع `@livekit/react-native` (ليس Expo Go). حالياً مؤشر
  التحدّث محاكاة مثل الويب.
- الألعاب داخل الغرفة (Ludo / Baloot / Jackaroo / Snake&Ladder).
- شاشات اللوبي الكاملة (المتجر، VIP، المسابقات، الأصدقاء…).

## السيرفر

`src/config.js` يشير افتراضياً إلى سيرفر Render:
`https://voice-room-server.onrender.com`. للتطوير محلياً، غيّره لعنوان جهازك
مثل `http://172.20.10.2:3001` وشغّل السيرفر من مجلد `server/`.
